La philosophie du libre dans la création numérique


Dans ses débuts le matériel informatique était vendu avec l’ensemble des logiciels, les deux formaient un ensemble. C’était le matériel qui assurait les revenus du fabricant, le logiciel servant juste à faciliter la vente en attirant la clientèle. En effet les personnes qui achetaient du matériel informatique possédaient souvent une équipe de développeurs, et les équipes entre elles s’échangeaient le code source de leurs logiciels.


Mais avec la démocratisation de l’informatique pour le grand public, des éditeurs de logiciels vont se spécialiser dans la vente de licences de logiciels. Pour ces logiciels il est souvent interdit d’analyser, d’améliorer et de redistribuer le code source.


Face aux restrictions que cela entraîne (impossible de personnaliser le logiciel à ses besoins, ou de corriger les bogues) des personnalités comme Richard Stallman ont décidé de se lancer dans la création de logiciels libres.
