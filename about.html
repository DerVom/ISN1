A propos


Concept est un site qui a pour vocation la partage, le partage d'idée, de connaissance, d'expérience, ...


Le site est d'abord conçu afin d'expérimenter les techniques apprises en cours d'ISN, il s'agit donc d'un site amateur. C'est un site pour le moment statique, elle n'est donc pas générée en fonction de l'utilisateur qui la consulte. En revanche le but est d'à terme construire un site dynamique nottament à l'aide de languages comme le PHP, le python ou le JavaScript.


Le créateur du site (Romain de Laage) a également pour vocation de promouvoir les logiciels libres, c'est pourquoi le code source du site sera public et disponible sous licence libre. Un dépot GitHub sera donc créé à cet effet. En attendant si vous voulez voir ses autres projets vous pouvez visiter son compte GitHub.


En attendant la suite et comme l'inspiration me manque tu peux utiliser ce formulaire de contact.
